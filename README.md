# Smuggler's Run

A space trading and combat game built with vanilla JavaScript and HTML5 Canvas.

## Description

Smuggler's Run is an engaging space trading game where players navigate through a vast galaxy, trade goods, upgrade their ship, and avoid or engage in combat with various factions. The game features dynamic economy, ship customization, and strategic gameplay elements.

## Features

- Dynamic space trading system
- Ship customization and upgrades
- Combat mechanics
- Multiple difficulty levels
- High score system
- Sound effects and background music
- Responsive design
- Minimap navigation

## Installation

1. Clone the repository:

```bash
git clone https://github.com/yourusername/smuglers-run.git
cd smuglers-run
```

2. Install dependencies:

```bash
npm install
```

3. Start the development server:

```bash
npm run dev
```

4. Build for production:

```bash
npm run build
```

## Controls

- Arrow Keys: Navigate ship
- Space: Fire weapons
- Tab: Toggle minimap
- ESC: Pause game
- Enter: Select menu options

## Development

The project uses Vite as the build tool and development server. The codebase is organized into several modules:

- `src/core/`: Core game logic and state management
- `src/managers/`: Game systems managers (Input, Audio, Render)
- `src/ui/`: User interface components
- `src/utils/`: Utility functions and helpers
